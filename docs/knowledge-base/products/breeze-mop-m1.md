# Breeze Mop M1 平板拖把商品详情

Breeze Mop M1 是面向家庭地面清洁的平板拖把，适合木地板、瓷砖、复合地板和日常湿拖场景。拖把头宽度为 38cm，拖布采用超细纤维材质，主打轻量、省力和低噪的手动清洁体验。

## 结构化参数

| 参数 | 值 |
|---|---|
| 型号 | BM-M1 |
| 品类 | 平板拖把 |
| 拖把头宽度 | 38cm |
| 拖布材质 | 超细纤维 |
| 杆体材质 | 铝合金伸缩杆 |
| 适用地面 | 木地板、瓷砖、复合地板 |
| 替换拖布型号 | BM-M1-PAD |
| 建议更换周期 | 3-6 个月，按磨损和清洁频率调整 |

## 适用场景

- 适合 60-120 平米家庭的日常湿拖和除尘。
- 适合木地板用户，使用时建议拧至微湿，避免长时间积水。
- 适合不想使用电动设备、希望低噪清洁的用户。

## 限制与注意事项

- 不建议用于大面积油污、施工灰或强腐蚀性清洁剂场景。
- 木地板使用后如果地面有明显水痕，应减少拖布含水量。
- 替换拖布必须使用 BM-M1-PAD 或明确标注兼容 BM-M1 的拖布。
